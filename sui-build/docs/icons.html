开发者文档

图标使用 icon font 字体实现。相比于图片，图标字体是矢量图，和普通文案一样可以非常方便的进行大小和颜色的调整。

  • 更新 v1.4.2：网页图标增加了几个新的图标。
  • 更新 v1.4.0：icomoon 的免费图标被移入 sui-append.css 中，如果有使用这些图标，注意引入这个文件。
  • 更新 v1.1.9：图标增加了两个 .icon-pc-undo .icon-pc-redo。
  • 更新 v1.1.8：小尺寸图标中增加了25个新的图标。
  • 更新 v1.1.7：增加了一个 .icon-pc-text。
  • 更新 v1.1.3：引入了了阿里巴巴图标库中的一组图标。
